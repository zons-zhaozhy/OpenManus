# 开发指南

## 开发环境设置
1. Python环境
   - Python 3.9+
   - 虚拟环境管理
   - 依赖管理

2. 开发工具
   - VSCode/PyCharm
   - Git
   - Docker（可选）

## 开发流程
1. 分支管理
   - main：主分支，保持与OpenManus同步
   - feature/*：功能开发分支
   - bugfix/*：问题修复分支
   - release/*：发布分支

2. 提交规范
   - feat：新功能
   - fix：修复
   - docs：文档更新
   - style：代码格式
   - refactor：重构
   - test：测试相关
   - chore：构建过程或辅助工具的变动

## 代码规范
1. Python代码规范
   - 遵循PEP 8
   - 类型注解
   - 文档字符串

2. 测试规范
   - 单元测试覆盖率 > 80%
   - 集成测试场景完整
   - 性能测试基准定义

3. 文档规范
   - 代码内文档完整
   - API文档及时更新
   - 示例代码充分
